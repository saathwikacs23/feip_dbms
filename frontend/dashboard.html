Platform Overview

Federated Architecture

This platform demonstrates a federated DBMS that integrates heterogeneous data sources without centralization.

Data Sources

  • PostgreSQL: Relational data
  • MongoDB: NoSQL data
  • CSV Files: Sensor readings

Query Engine

Apache Drill acts as the federated layer, allowing ANSI SQL queries across all connected data sources.
